# PoseMind - AI 动作分析系统

PoseMind 是一个基于 AI 的动作分析系统，可以帮助用户分析各种运动动作的正确性，包括但不限于：
- 篮球投篮动作
- 瑜伽姿势
- 健身动作

## 功能特性

- 🎯 实时动作捕捉和分析
- 📊 动作评分和建议
- 🎓 动作指导和建议
- 📱 支持网页端访问
- 🔄 实时反馈

## 技术栈

### 后端
- Python
- FastAPI
- OpenCV
- MediaPipe
- PyTorch

### 前端
- Next.js
- TypeScript
- Tailwind CSS
- React Query

## 开发环境设置

### 后端设置
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
.\venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

### 前端设置
```bash
cd frontend
npm install
npm run dev
```
